# 머신러닝 기반 사고 감지 시스템 

스마트폰의 카메라와 센서 데이터를 활용하여 실시간으로 차량 충돌 등의 사고를 감지하고, 해당 정보를 백엔드 서버로 전송하는 크로스 플랫폼(Cross-platform) 모바일 애플리케이션입니다.

# app 빌드
``` shell
flutter pub get
flutter run
```
